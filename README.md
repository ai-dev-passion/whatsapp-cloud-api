`whatsapp-cloud-api` is an unofficial library for creating bots and sending/receiving messages using the Whatsapp Cloud API.

## Install

```bash
npm i whatsapp-cloud-api
```

## Development

```bash
# install npm modules
npm i

# eslint
npm run lint

# typescript check
npm run ts-check

# test
npm t

# build
npm run build
```
